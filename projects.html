DNN Accelerator on DE1-SoC

Designed a hardware neural-network accelerator on the DE1-SoC FPGA using Verilog and a Nios II soft-core processor. Implemented fixed-point matrix multiply, ReLU activation, and SDRAM double-buffering to classify MNIST digits entirely in hardware.

Self-Balancing Robot

Built a two-wheeled inverted-pendulum robot with an Arduino Nano 33 BLE Sense, MPU-6050 IMU, and dual motor encoders. Tuned cascaded PID loops (angle + velocity), added a BLE remote-control interface, and displayed live telemetry on an I²C LCD.

Metal-Detecting Rover

Designed and assembled an autonomous RC rover that sweeps terrain for buried metal objects. An inductive proximity sensor feeds an Arduino Mega which drives servo steering and DC motors, with an LCD + buzzer alert when a detection threshold is crossed.
